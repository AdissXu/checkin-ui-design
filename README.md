# 签到页面 UI 设计稿

基于电商首页风格的签到页面设计。

## 预览

在线预览：[点击这里](https://adissxu.github.io/checkin-ui-design/checkin-ui.html)

## 设计特点

- **配色方案**：青绿色主色调 `#40E0D0`，与电商首页保持一致
- **圆角风格**：16px 大圆角，现代简洁
- **字体**：PingFang SC / SF Pro Display
- **布局**：iPhone 尺寸适配，375×812px

## 文件说明

- `checkin-ui.html` - 可交互的 HTML 演示稿
- `checkin-ui-spec.md` - 详细设计规范文档

## 页面结构

1. **顶部统计卡片** - 显示连续签到天数和累计积分
2. **今日签到按钮** - 主要操作入口
3. **签到日历网格** - 7天签到状态展示
4. **终极奖励卡片** - 第7天特殊奖励
5. **规则说明** - 签到规则列表

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- 纯原生实现，无依赖
