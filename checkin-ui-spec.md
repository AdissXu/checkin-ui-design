# 签到页面 UI 设计稿

## 设计规范（基于首页风格）

### 配色方案
- **主色（品牌色）**: `#40E0D0` (青绿色/薄荷绿)
- **主色渐变**: `#40E0D0` → `#20B2AA`
- **背景色**: 
  - 顶部: `#F8F9FC` (极浅灰蓝)
  - 卡片背景: `#FFFFFF` (纯白)
  - 签到卡片背景: `#F0FFFE` (极浅青)
- **文字色**:
  - 主文字: `#1A1A1A` (近黑)
  - 次要文字: `#666666` (中灰)
  - 辅助文字: `#999999` (浅灰)
- **功能色**:
  - 成功/已签到: `#40E0D0`
  - 可领取: `#40E0D0`
  - 未开启: `#E8E8E8`
  - 终极奖励: `#FFD700` → `#FFA500` (金橙渐变)

### 字体规范
- **字体**: PingFang SC / SF Pro Display
- **标题**: 20px, Bold, `#1A1A1A`
- **天数标签**: 14px, Medium, `#1A1A1A`
- **按钮文字**: 14px, Medium, `#FFFFFF`
- **辅助文字**: 12px, Regular, `#999999`

### 圆角规范
- **大卡片**: 16px
- **签到格子**: 12px
- **按钮**: 20px (胶囊形)
- **标签**: 8px

### 间距规范
- **页面边距**: 16px
- **卡片间距**: 12px
- **内部间距**: 16px

## 交互状态

### 已签到状态
- 背景: 填充主色 `#40E0D0`
- 图标: 白色对勾 ✓
- 文字: 白色 "已签到"

### 可签到状态
- 背景: 浅青 `#F0FFFE`
- 边框: 1px 主色 `#40E0D0`
- 图标: 主色箭头 →
- 文字: 主色 "点击领取"
- 点击效果: 轻微缩放 + 阴影

### 未开启状态
- 背景: 浅灰 `#F5F5F5`
- 图标: 灰色问号 ?
- 文字: 灰色 "未开启"
- 不可点击

## 与首页风格一致性

| 元素 | 首页风格 | 签到页应用 |
|------|---------|-----------|
| 主色 | 青绿 `#40E0D0` | 完全一致 |
| 圆角 | 16px 大圆角 | 卡片 16px，格子 12px |
| 字体 | PingFang SC | 完全一致 |
| 按钮 | 胶囊形，圆角 20px | 完全一致 |
| 卡片 | 白色背景，柔和阴影 | 完全一致 |
| 网格 | 双列布局 | 三列布局（功能需要） |
| 标签 | 紫色促销标签 | 改为青绿色状态标签 |

---

## 视觉层次

1. **第一层**: 顶部统计卡片（渐变背景，最醒目）
2. **第二层**: 今日签到按钮（CTA，吸引点击）
3. **第三层**: 签到日历网格（核心功能）
4. **第四层**: 终极奖励卡片（特殊奖励，金色突出）
5. **第五层**: 规则说明（辅助信息）

整体保持首页的清新、简洁、现代感，同时融入签到功能的游戏化元素。
